Что нужно для запуска:

1.Настройки и установка утилит

Для работы проекта нужен Python 3.14.2.
И Библиотеки

Впишите строки ниже в терминал Visual Studio (я пользовался именно ей, а не VS Code):

pip install vosk          # Сердце (распознавание речи)
pip install sounddevice   # Слух (работа с микрофоном)
pip install pyttsx3       # Голос (синтез речи)
pip install gigachat      # Мозг (интеллект и общение)
pip install colorama      # Красота (цветной вывод в консоли)

2. Голосовой модуль

Установите модель Vosk. Для русского языка я использовал small версию.
(Не переживайте за место на диске, она весит всего 25-45 МБ). Папку с моделью нужно переименовать в model и положить в корень проекта.


3. Ии(GigaChat)

Очень важная деталь: для работы ИИ я использовал бесплатные API-ключи от GigaChat (это ИИ от Сбера).

1.  Погуглите, где и как взять их ключи (они бесплатны).

2.    Как только получили ключ, найдите в начале кода строку auth_key = "...".

3.   Вставляйте туда свой ключ, и всё должно заработать
